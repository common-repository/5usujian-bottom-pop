== Description ==
搭配另一款【弹窗艺术家插件】可以实现<strong>文件上传、表单提交并在线支付、文件上传、合同签约和手写签字</strong>等特色功能；
<strong>弹窗艺术家插件效果演示：<a href="https://www.5usj.cn/wp/wysj-popup" target="_blank">https://www.5usj.cn/wp/wysj-popup</a></strong>
留言记录保存在后台，并支持邮箱提醒；
提交表单并在线支付，实现在线预约、预售等功能；
自定义弹窗触发方式，例如进入页面弹窗、离开页面弹窗、用户点击弹窗等等；
自定数学公式，根据用户输入内容自动计算结果，例如房屋装修费用计算器；
自定条件逻辑，根据用户输入控制某些字段显示或隐藏；
支持导入弹窗设置和导出弹窗留言记录为pdf和csv，自动生成统计折线图，方便地分析弹窗运营效果；

5usujian bottom pop表单弹窗插件可以在wordpress网站底部添加一个优雅实用的表单悬浮窗或嵌入式表单；
支持拖拽式定制表单，表单类型包含单行文本、多行文本、单选下拉框、多选下拉框、单选按钮、日期选择器、城市选择器；
客户提交的资料保存在后台，你可以在后台对客户留言进行查阅、删除、批量处理等操作，客户留言也可以以邮件的形式发送至邮箱；
插件支持2种模式，一种为底部悬浮窗，支持添加广告图片，让表单更加美观醒目；一种为嵌入式表单，将表单嵌入某个页面，变为一个普通的留言表单；
插件为响应式架构，完美适配手机端；
支持全方位的样式修改，包括按钮颜色、表单宽度、背景图片、广告图片等等，让你的设计随意发挥。

== Installation ==
1. 插件-安装插件-上传插件压缩包，启用或者将插件文件上传至“/wp-content/plugins/”解压，然后在插件中启用；
2. 点击左侧菜单的“客户咨询列表->底部弹窗设置”进入设置；

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
